# Inbox leadów

Świeże zapytania do obróbki. Po przygotowaniu odpowiedzi → przenoszę dane do `crm.csv`.
Format jednego wpisu poniżej (kopiuję na górę przy każdym nowym leadzie).

---

## [DATA] — Imię / źródło (Otodom / OLX / FB / mail)
- **Oferta, której dotyczy:** {nazwa teczki w nieruchomosci/ lub „ogólne"}
- **Kontakt:** {email / telefon}
- **Treść zapytania:** {co napisał klient}
- **Typ:** kupujący / sprzedający / najem
- **Temperatura:** gorący / ciepły / zimny
- **Draft odpowiedzi:** {link do pliku w teksty/ lub status „do przygotowania"}
- **Status:** nowy

---

_(Brak leadów — powiedz „sprawdź maile", a pobiorę i wypełnię tę listę.)_
