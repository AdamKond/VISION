# Szablon: odpowiedź na lead + sekwencja follow-up

Zasada: zawsze tworzę **draft** do akceptacji. Personalizuję wg pytania klienta i `oferta.md`.

## Pierwsza odpowiedź (po zapytaniu z portalu)
```
Dzień dobry {imię},

dziękuję za zainteresowanie ofertą {nazwa/oferta} ({metraż} m², {dzielnica}).

Odpowiadając na Pana/Pani pytanie: {konkretna_odpowiedź}.

Najważniejsze atuty:
✓ {atut_1}
✓ {atut_2}
✓ {atut_3}

Chętnie pokażę mieszkanie na żywo — czy pasuje Panu/Pani {propozycja_terminu_1}
lub {propozycja_terminu_2}? Mogę też wysłać film z prezentacją mieszkania.

Pozdrawiam,
{agent} | {telefon}
```

## Segmentacja leada (do crm.csv)
- **typ**: kupujący / sprzedający / najem
- **temperatura**: gorący (chce oglądać) / ciepły (pyta o szczegóły) / zimny (ogólne)
- **status**: nowy → kontakt → oglądanie → negocjacje → zamknięty

## Sekwencja follow-up (proponuję, Ty zatwierdzasz wysyłkę)
- **Dzień +2** (brak odpowiedzi): „Czy oferta jest nadal aktualna dla Pana/Pani? Mogę
  podesłać dodatkowe zdjęcia lub film."
- **Dzień +5**: dorzucenie wartości — film/staging/porównanie z okolicą.
- **Dzień +10**: miękkie domknięcie — „Daję znać, że mam zainteresowanych; jeśli temat
  wciąż aktualny, zarezerwuję dla Pana/Pani termin oglądania."

## Po oglądaniu
„Dziękuję za wizytę. Jak wrażenia? Chętnie odpowiem na pytania i — jeśli oferta pasuje —
podpowiem kolejne kroki (rezerwacja, finansowanie)."
